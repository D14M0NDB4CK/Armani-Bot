# Privacy Policy 

This Privacy Policy explains how Armani Bot ("the Bot", "we", "us", or "our") collects, uses, stores, and protects information when users interact with the Bot on Discord.

##### *By using Armani Bot, you agree to the collection and use of information in accordance with this Privacy Policy.*

## Information We Collect

Armani Bot may collect and store the following information:

* User Information
* Discord User IDs
* Usernames and display names
* Server-specific settings and preferences
* Server Information
* Discord Server (Guild) IDs
* Channel IDs
* Role IDs
* Configuration settings required for bot functionality
* Message Information

Armani Bot does not permanently store message content unless required for a specific feature, such as:

* Moderation logs
* Ticket systems
* Message logging features enabled by server administrators
* Usage Data
* Command usage statistics
* Error logs
* Feature usage data necessary for improving the Bot

## How We Use Information

Collected information may be used to:

* Provide and maintain bot functionality
* Store server configuration settings
* Process commands and interactions
* Provide moderation and logging features
* Improve performance and reliability
* Diagnose bugs and technical issues

## Data/Content

### Sharing

**We do not sell, rent, or trade user data.**

Information may only be shared:

* When required by law

### Retention

Data is retained only as long as necessary to provide the Bot's services.

Server administrators may request removal of server-specific data by removing the Bot from their server or contacting us directly.

Reasonable measures are taken to protect stored information from unauthorized access, disclosure, alteration, or destruction. However, no method of electronic storage is completely secure.
